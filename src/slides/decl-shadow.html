This is also retro-compatible, meaning that re-declaring a shadow root with .attachShadow() will just remove the one created by the Declarative Shadow DOM, letting old Web Components still work.

  Bring the `80s back!

DSD also makes it easy (again) to load stylesheets for Web Components. All you need to do is to insert a <link> element inside the template, as we used to do with JavaScript. The stylesheet get cached too, so it won't be loaded again for all the other instances of the same component. Declarative Shadow DOM
