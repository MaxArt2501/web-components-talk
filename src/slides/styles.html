The >>> operator…

  display: none;
}

… would defeat the purpose of style encapsulation

Only this works…

<my-component style="--my-color: blue"></my-component>
/* my-component.css */
:host > div {
  background-color: var(--my-color);
}

A great idea!

/* styles.css */
:root {
  --my-mixin: {
    background-color: red;
    color: white;
  }
}
/* my-component.css */
:host > div {
  @apply(--my-mixin);
}

… but what does this even mean?

:root {
  --my-color: red;
  --my-mixin: {
    background-color: var(--my-color);
  }
}
.a-test {
  --my-color: green;
  @apply(--my-mixin);
}
